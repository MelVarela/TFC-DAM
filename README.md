# Notas & Mazmorras
## Guía de despliegue del proyecto
**ADVERTENCIA**
El proyecto todavía se encuentra en fase de desarrollo, por lo que todavía no tiene ninguna build desplegable.
### Requisitos
Actualmente, para poder acceder al proyecto, es necesario tener Android Studio instalado en el equipo, así como Git si se quiere clonar el proyecto
### Obtención del código
Para descargar el código del proyecto a tu equipo puedes descargarlo directamente desde el apartado *code* en GitHub, o bien puedes clonar el proyecto con el comando `git clone https://github.com/MelVarela/TFC-DAM.git` en la carpeta donde quieras copiar el proyecto.  
**Ten en cuenta que actualmente el código del proyecto se encuentra en la rama *app_android*, por lo que será necesario que uses el comando `git checkout app_andoid` una vez clonado, o que cambies a la rama app_android antes de descargarlo desde GitHub.**
### Ejecución
Una vez obtenido el código, abre el proyecto que se encuentra dentro de la carpeta *APP* en Android Estudio y ejecútalo en un emulador o en la proyección en tu dispositivo.
